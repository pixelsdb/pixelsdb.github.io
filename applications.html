Applications

Systems built on top of PixelsDB

PixelsDB extends beyond the core engine into elastic query processing, web-based analytics, freshness services, and serverless acceleration.

Pixels-Turbo

Turbo provides serverless elastic query processing, combining autoscaling execution with serverless acceleration for bursty analytical workloads.

Explore Pixels-Turbo

Pixels-Rover

Rover delivers a web-based analytics interface with flexible SLAs, interactive query workflows, and natural-language-assisted exploration.

Open Pixels-Rover

Pixels-Retina

Retina supports freshness and ACID transactions, enabling real-time synchronization and consistent analytical access on evolving data.

See Retina in the main project

Pixels-Spike

Spike is the serverless computing framework used to accelerate query processing bursts and support the elastic execution model of PixelsDB.

Browse Pixels-Spike

Pixels-Sink

Sink services capture upstream changes and apply them into Pixels-backed storage for fresh analytical views.

Open Pixels-Sink

Connector ecosystem

Integrations with Trino, PrestoDB, Hive, Spark, PostgreSQL, and others make PixelsDB usable across existing analytical stacks.

View integration references
