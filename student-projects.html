Student Projects

Teaching, coursework, and student-led exploration

The PixelsDB ecosystem also includes educational systems, course material, and student projects that extend or study modern open data systems.

mini-pixels

An educational analytic storage engine derived from Pixels, designed to make the core ideas more approachable for teaching and learning.

Open mini-pixels

osds-course

Open-source data systems course material that can be used to teach systems concepts, implementation techniques, and related project work.

Browse the course repository

cdw-user-study

A student-oriented user study on pricing models and related features of cloud data warehouses.

See the user study project

elastic-bench

Benchmark tooling that generates workloads for elasticity evaluations and experimentation with cloud analytical systems.

Open elastic-bench

cloud-container-benchmark

Benchmark experiments for cloud containers such as AWS Fargate and AWS Lambda, useful for systems coursework and evaluation projects.
